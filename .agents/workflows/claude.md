This workflow helps you quickly stage, commit, and push your changes to the master branch.

1. Stage all changes:
// turbo
`git add .`

2. Commit changes with a message:
`git commit -m "update: documentation and layout improvements"`

3. Push to remote:
// turbo
`git push origin master`

4. Open the website to verify changes:
// turbo
`open https://kprinceing.github.io/person-website/`
